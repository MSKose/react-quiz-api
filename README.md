## How To Use

```bash
# Clone this repository
$ git clone https://github.com/MSKose/react-quiz-api.git

# Install dependencies
    $ yarn

# Run the app
    $ yarn start

# Run your Back-End on localhost 8000
    # see the Back-End repo
```
